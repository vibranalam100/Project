# Automobile Price Prediction ML Project

Intermediate-level machine learning project using the UCI Automobile Dataset.

## Dataset

The project uses the UCI Automobile dataset.

The dataset contains automobile specifications such as:

- make
- fuel type
- body style
- engine size
- horsepower
- city MPG
- highway MPG
- insurance risk rating
- normalized losses
- price

## Project Features

- Download real automobile data from the internet
- Clean missing values
- Handle categorical and numerical features
- Feature engineering
- Exploratory data analysis
- Data visualizations
- Model comparison
- Cross validation
- Final model saving
- Price prediction script

## Libraries Used

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- joblib
- requests
